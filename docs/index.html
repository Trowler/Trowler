Trowler

A universal scaffolding tool.

How it works?

Open your terminal, run the magic command, done.

Customizable

Trowler can be customized using custom commands that will be runned while generating your project

Installation

Install, run, done. Simple as that.

# Copy and Paste the following green line in a terminal.
$ npm i -g trowler
/usr/local/bin/trowler -> /usr/local/lib/node_modules/trowler/cli/cli.js 
+ trowler@1.0.0
updated 1 package in 1s
$ trowler_
